用途： 记录每日要说明的内容和存在的问题。
用法： 如果某一天有要说明的内容或者存在问题。请新建一个当日的log。
文件命名规则： date_mark.txt
	date为日期，MM-DD
	mark为名字缩写：
		SR 苏睿聪
		TS 檀森
		YD 姚E
文件书写规则：
文件分为两部分：
第一部分为说明，用“----说明----”表示

第二部分为问题，用“----问题----”表示，问题中需要指明问题提交给谁：
分块表示 
	“----SR----” 表示提交给苏睿聪
	“----TS----” 表示提交给苏睿聪
	“----YD----” 表示提交给苏睿聪

请务必每日查看DailyLog文件夹，直到project完成。
请在当日解决前一日DailyLog记录下的问题，并在其问题上标记。
